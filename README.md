# AI Customer Analytics Solution

## Overview
AI-powered customer analytics platform built using Python, Flask, Machine Learning, and SQLite.

## Features
- Customer Purchase Prediction
- Customer Segmentation
- Product Recommendation System
- Interactive Dashboard
- Prediction History
- Reports Generation
- User Login & Registration

## Technologies Used
- Python
- Flask
- Scikit-Learn
- Pandas
- Matplotlib
- SQLite
- Bootstrap
